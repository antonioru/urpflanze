Creating a scene

Hello Rect!

After installing the library, you only need three classes for create a scene:
  • Scene Reference to the scene being created
  • Rect A primitive shape to add to the scene
  • DrawerCanvas Class that allows you to "draw" the scene on canvas

Repetitions

With Urpflanze you can manage two types of repetitions: ring or matrix.

Ring

To define a repetition Ring just set a numeric value to the repetitions property. The shapes will be distributed over the entire circumference whose radius is defined by the distance property and will be rotated towards the center.

Matrix

To define a repetition to Matrix just set an array of one or two numbers to the repetitions property. If only one value is entered (eg [3]) the matrix will have 3 rows and 3 columns. If, on the other hand, two numbers are passed, for example [3, 5], the matrix will have 3 rows and 5 columns.

For this type of repetition it is possible to set distance property as the distance between the rows and columns starting from the center of the scene.

Manage the scene

Before continuing it is important to explain the hierarchy of classes that can be added to the scene.
  • Group is a logical class, a SceneChild container, its purpose is to propagate its properties to children
  • Shape is a SceneChild encapsulator on which properties can be applied
  • ShapeBuffer is a ShapePrimitive generated by a static buffer
  • ShapeLoop is a ShapePrimitive generated by a loop
  • ShapeRecursive is a shape that repeats a SceneChild on each vertex and it is possible to apply properties
You can basically use some ShapeBuffer and ShapeLoop
  • ShapeBuffer Line, Triangle, Rect
  • ShapeLoop Polygon, Circle, Spiral, Rose, Lissajous

Containers: Group and Shape

With Groups, you can apply the same properties to multiple SceneChilds.

With the Shape instead it is possible to repeat and transform the whole SceneChild.

Styling

For the ShapePrimitive - which are the shapes that will be drawn - it is possible set the border or fill color through the style property.
