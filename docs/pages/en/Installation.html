Installation

Install from CDN

The most immediate way to include Urpflanze in your project is to use an online hosted version. 
<!-- Link to the latest minified full version -->
<script src="https://cdn.jsdelivr.net/npm/urpflanze"></script>
<!-- Customizable link -->
<script src="https://cdn.jsdelivr.net/npm/urpflanze[@version]/build/urpflanze[-light][.min].js"></script>
Full version All versions

Download

If you prefer, you can download the full version, save it somewhere in your project and include it in your code. 
<!-- Full version -->
<script src="path-to/urpflanze.js"></script>

<!-- Minified version-->
<script src="path-to/urpflanze.min.js"></script>
Full version Minified full version
If you are starting out and don't need all the features you can download the lite version 
<!-- Light version -->
<script src="path-to/urpflanze-light.js"></script>

<!-- Minified light version-->
<script src="path-to/urpflanze-light.min.js"></script>
Light version Minified light version

Install from npm

To install it just run the command: 
npm i urpflanze --save
After the installation is complete, we will be able to use Urpflanze in your code 
/**
 * Full importing
 */
import * as Urpflanze from 'urpflanze'

const scene = new Urpflanze.Scene()

/**
 * Selective import
 */
import { Scene } from 'urpflanze'

const scene = new Scene()
To include the light version use urpflanze/dist/index-light
