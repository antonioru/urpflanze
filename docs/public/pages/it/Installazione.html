Installazione

Inclusione tramite download

Il modo più immediato per includere Urpflanze nel tuo progetto è scaricare la versione completa, salvarla da qualche parte nel tuo progetto ed includerla nel tuo codice. 
<!-- Versione completa -->
<script src="path-to/urpflanze.js"></script>

<!-- Versione minificata-->
<script src="path-to/urpflanze.min.js"></script>
Versione completa Versione completa minificata
Se sei agli inizi e non hai bisogno di tutte le funzionalità puoi scaricare la versione lite 
<!-- Versione light -->
<script src="path-to/urpflanze-light.js"></script>

<!-- Versione light minificata-->
<script src="path-to/urpflanze-light.min.js"></script>
Versione light Versione light minificata

Inclusione tramite CDN

Se lo preferisci, puoi utilizzare una versione hostata online. 
<!-- Link customizzabile -->
<script src="path-to/urpflanze[-light][.min]@[version].js"></script>

Installazione tramite npm

Siccome la libreria è hostata nei registry di Github, prima di installarla bisognerà creare un file nella directory contenente il file package.json denominato .npmrc ed aggiungere la seguente riga 
registry=https://npm.pkg.github.com/genbs
Dopo la creazione sarà possibile per npm trovare la libreria. Per installarla basterà lanciare il comando: 
npm i --save @genbs/urpflanze
Al termine potai includere Urpflanze nel tuo codice 
/**
 * Importazione completa
 */
import * as Urpflanze from '@genbs/urpflanze'

const scene = new Urpflanze.Scene()

/**
 * Importazione selettiva
 */
import { Scene } from '@genbs/urpflanze'

const scene = new Scene()
Per includere la versione light utilizza @genbs/urpflanze/dist/index-light
